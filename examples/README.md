# Examples for MiMMO (Minimal Memory Manager for Openacc)

In this directory you will find some examples of how to use MiMMO in a C++ code.

Instructions to compile and run each example can be found in the `compile_and_run.txt`
file placed in the related sub-directory.

The following examples are available:
- `scalar_product`: basic example of accelerated scalar poduct using MiMMO as
  memory manager;
- `globals`: example of dual memory manager usage with global variables.
